                     ❓ SOMETHING NOT WORKING?
================================================================================

Issue: MongoDB not installed
→ Download: mongodb.com/try/download/community
→ Run installer and choose "Install as Service"

Issue: Backend won't start
→ Check: MongoDB is running (Windows Services)
→ Try: cd NEETP\backend && npm run dev

Issue: Frontend blank page
→ Press F12 (DevTools)
→ Check Console for errors
→ Make sure backend is running

Issue: Port already in use
→ Edit backend/.env
→ Change PORT=5000 to PORT=5001

Issue: Still having trouble?
→ Read: QUICK_REFERENCE.md (troubleshooting section)
