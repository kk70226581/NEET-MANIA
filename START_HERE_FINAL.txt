================================================================================
                           TROUBLESHOOTING
================================================================================

Issue: MongoDB won't install
→ Download from: mongodb.com/try/download/community

Issue: Backend won't start
→ Check MongoDB is running (Windows Services)
→ Check port 5000 is free
→ Run: npm install

Issue: Frontend won't start
→ Try: npm install
→ Try: npm start
→ Wait for compilation (first time takes 1-2 min)

Issue: Blank page at localhost:3000
→ Press F12 (DevTools)
→ Check Console for errors
→ Make sure backend is running
